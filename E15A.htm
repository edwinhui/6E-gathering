Answer to Q15.

15. What are the following abbreviations in full?

  1. E.g. => EXEMPLI GRATIA
  2. I.e. => ID EST
  3. N.B. => NOTA BENE
  4. P.S. => POSTSCRIPT
  5. Etc. => ET CETERA
  6. Et. Al. => ET ALIA
  7. C.f. => CONFER
  8. Q.E.D. => QUOD ERAT DEMONSTRANDUM
  9. V.S. => VERSUS
